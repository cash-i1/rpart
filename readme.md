## Compilation
1. git clone it 
2. cd into the directory and run `make`
3. it should launch

## Adding new particles
1. git clone it 
2. cd into the directory
3. go to particles/particle_ref.cpp
4. copy one of the ones already there (like sand or something)
5. change the values
6. run `make`
7. it should work
